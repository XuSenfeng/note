## 代码

```c#
using System;
using System.Collections.Generic;
using System.Linq;
using System.Text;
using System.Threading.Tasks;

namespace Hello
{
    class Program
    {
        static void Main(string[] args)
        {
            Console.WriteLine("这是我的程序");
            Console.ReadKey();
        }
    }
}
```

注释//

### 结构

```c#
//命名空间
namespace Hello
{
    //类型
    class Program
    {
        //函数方法
        static void Main(string[] args)
        {
            Console.WriteLine("这是我的程序");
            Console.ReadKey();
        }
    }
}
```

```c#
//类型, 名字自定义, 不能是数字名字开头
class Program
{
    //函数方法, 程序入口
    static void Main()
    {
    }
}
```
