# day4C语言以及显示

```assembly
[INSTRSET "i486p"]				; 486这个程序的应用位置, nask使用的指令

_write_mem8:	; void write_mem8(int addr, int data);
		MOV		ECX,[ESP+4]		; [ESP+4]存放的是地址
		MOV		AL,[ESP+8]		; [ESP+8]存放的是数据
		MOV		[ECX],AL
		RET
```

>   在使用C语言的时候可以使用的寄存器有EAX, ECX, EDX
