# 基础概念

[主页 - Ultralytics YOLO 文档](https://docs.ultralytics.com/zh/)

+ V1: 在识别一个物体的时候, 首先提取物体的特征值, 将输入图片（例如448x448像素）均匀划分为 **S x S** 个网格, 依次遍历一下这个图像, 在图片里面计算物体的识别概率

> 每个边界框需要预测5个值：
>
> 1. `(x, y)`：**框的中心点**相对于**当前格子**的坐标。
> 2. `(w, h)`：**框的宽度和高度**相对于**整张图片**的比例。
> 3. `confidence`（置信度）：这个框包含一个物体的把握有多大？以及框得准不准？公式是：`Pr(Object) * IOU`。如果格子中没有物体中心，置信度应为0

+ V2: 同一个特征值对应多个窗口, 以前每个格子预测的框形状是随便学的。现在，它先看看训练数据里最常见的物体形状（比如人瘦高、汽车扁长），记住5种常见的“框模板”。预测时，它就在这些模板基础上微调, 用k-means聚类算法对数据集中所有真实框的宽高进行聚类

+ V3: 使用不同的网络去识别大小不同的物体, 网络在**三个不同尺度**上进行检测, 最后会得到三种不同尺寸的特征图：**大的（如52x52）、中的（26x26）、小的（13x13）**

+ V4: 细节处理, 在之前的处理的时候, 依次处理一张图片, 现在处理多张图片拼在一起, 同时使用DropBlock随机去除一些点之后训练, 现在直接去除一个区域, 同时在之前定义标签的时候使用的是0和1, 现在把标签改为一个范围值, 避免过拟合

    还有加入注意力机制, 用于处理不同的窗口之间的关系, 实际使用的时候不只是使用自己的特征, 同时一个物体不同的位置的权重应该是不同的, 一般来说更加注重物体的边界

+ V5: V4的工程版项目

+ V6: 基本和V5一样

+ V7: 推理提速, 统一卷积核的大小, 1x1的卷积可以在外面补0, 统一为3x3的

+ V8: 一个优化比较好的基础模型, Yolo是一个特征提取的框架, 实际提取出来的信息可以做不同的处理达到不同的效果(检测, 分割, 姿态...)

+ V9: 在训练的时候, 由于梯度传递是反向的, 所以后面的层学习的比较好 ,所以在比较浅的层连接到输出

+ V10: 提升速度

## v8及以后

基础环境里面需要使用torch

```bash
pip install torch torchvision torchaudio jupyterlab
```

yolo的python包在新版本改为使用ultralytics

```bash
# Clone the ultralytics repository
git clone https://github.com/ultralytics/ultralytics

# Navigate to the cloned directory
cd ultralytics

# Install the package in editable mode for development
pip install -e .
```
