
# QQ机器人

## 安装

```bash
docker run -d \
  --name snowluma \
  --restart unless-stopped \
  --shm-size=1g \
  --cap-add=SYS_PTRACE \
  --security-opt seccomp=unconfined \
  -e VNC_PASSWD=vncpasswd \
  -e SNOWLUMA_WEBUI_PORT=5099 \
  -e SNOWLUMA_QQ_FLAGS="--disable-gpu --disable-software-rasterizer --disable-gpu-compositing" \
  -p 5900:5900 \
  -p 6081:6081 \
  -p 5099:5099 \
  -p 3000:3000 \
  -p 3001:3001 \
  -v snowluma-data:/app/snowluma-data \
  -v snowluma-qq-config:/app/.config \
  -v snowluma-qq-data:/app/.local/share \
  motricseven7/snowluma:latest
```

访问本地 http://localhost:6081/ 默认密码  `vncpasswd`
控制台界面在 http://127.0.0.1:5099/ 使用下面的命令获取初始密码
```bash
(base) jiao@jiaodeMacBook-Air smowluma % docker logs snowluma 2>&1 | grep -E "临时密码|initial credentials" | tail -n 1

13:43:07 INFO               [WebUI] initial credentials: user=admin password=808fa8c8015a2292
```
