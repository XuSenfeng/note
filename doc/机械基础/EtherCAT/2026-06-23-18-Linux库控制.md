# LInux库控制

实际使用的库是 `libethercat`

## 控制流程

1. 使用命令启动服务, 开启ethernet设备
2. 初始化申请一个`master`
3. 创建`domain`的实际数据发送使用的内存空间
4. 创建从站的数据句柄
5. 注册 PDO 条目
6. 激活主站
7. 获取实际的映射指针
8. -循环-
9. 接受数据
10. 处理数据
11. 写入控制字
12. 输出排入队列
13. 发送
14. -结束循环-

## 示例

```c
#include <ecrt.h>
#include <unistd.h>

/* ───── 1. 全局句柄 ───── */
static ec_master_t  *master    = NULL;   // 主站
static ec_domain_t  *domain    = NULL;   // 过程数据域
static uint8_t      *domain_pd = NULL;   // 域的进程映像指针

/* 从站身份 (按你的设备改) */
#define ALIAS       0
#define POSITION    0                 // 总线位置
#define VENDOR_ID   0x00000168
#define PRODUCT_ID  0x00000011

/* ───── 2. 要交换的 PDO 条目, 偏移会被自动填进这两个变量 ───── */
static unsigned int off_ctrl;         // 0x6040 控制字 (输出)
static unsigned int off_status;       // 0x6041 状态字 (输入)

/* ───── 3. 注册表: "把这些对象登记到 domain, 偏移写回 offset 变量" ───── */
const static ec_pdo_entry_reg_t domain_regs[] = {
    {ALIAS, POSITION, VENDOR_ID, PRODUCT_ID, 0x6040, 0x00, &off_ctrl},
    {ALIAS, POSITION, VENDOR_ID, PRODUCT_ID, 0x6041, 0x00, &off_status},
    {}                                // 结束标记(全0)
};

int main(void)
{
    /* ① 请求主站 (索引0) */
    master = ecrt_request_master(0);
    if (!master) return -1;

    /* ② 创建过程数据域 */
    domain = ecrt_master_create_domain(master);
    if (!domain) return -1;

    /* ③ 获取从站配置句柄 */
    ec_slave_config_t *sc =
        ecrt_master_slave_config(master, ALIAS, POSITION, VENDOR_ID, PRODUCT_ID);
    if (!sc) return -1;

    /* ④ 注册 PDO 条目 (算偏移)， 实际的偏移记录在给的指针里面 */
    if (ecrt_domain_reg_pdo_entry_list(domain, domain_regs)) return -1;

    /* ⑤ 激活主站 (之后不能再改配置) */
    if (ecrt_master_activate(master)) return -1;

    /* ⑥ 拿到进程映像指针 */
    domain_pd = ecrt_domain_data(domain);
    if (!domain_pd) return -1;

    /* ⑦ 实时循环 */
    while (1) {
        ecrt_master_receive(master);   // 收: 把网线上回来的数据搬进映像
        ecrt_domain_process(domain);   // 处理输入

        /* —— 在这里读写过程数据 —— */
        uint16_t status = EC_READ_U16(domain_pd + off_status);  // 读状态字
        EC_WRITE_U16(domain_pd + off_ctrl, 0x000F);             // 写控制字
        (void)status;

        ecrt_domain_queue(domain);     // 把输出排进队列
        ecrt_master_send(master);      // 发: 把映像发到网线

        usleep(1000);                  // 1ms 周期
    }
    return 0;
}
```
