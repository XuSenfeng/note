
# 220V-380V区别

就是相电压和线电压的区别
220V是相电压，380V是线电压，但这并不是它们不一样的原因，因为在三角形接法的三相电路中，相电压和线电压是相等的。

## 相电压线电压

入户电压为220V的用户，称为单相用户，其入户线路有两根，分别为火线与零线。所谓“单相”，其实是指单根相线，这个相线就是我们常说的火线，因为它一直带有电压
![image.png](https://picture-01-1316374204.cos.ap-beijing.myqcloud.com/lenovo-xiangong/20260701113947869.png?imageSlim)

火线（或者说相线）与零线（或者说中性线）之间的电压称为相电压，**火线与火线之间的电压**称为**线电压**，但这是基于存在零线的情况下的。如果三相线路采用三角形接法时，那么它是没有零线（中性线）的
![image.png](https://picture-01-1316374204.cos.ap-beijing.myqcloud.com/lenovo-xiangong/20260701114000876.png?imageSlim)

三相电路中，三相电源和三相负载中每一相的电压称为相电压，各输电线的线端之间的电压，称为线电压
单相用户，就可以看成三相电路中的每相负载, 配电变压器低压侧引出四根线， 每个单相用户都选择三根火线中的一根分接，而零线都是从总零线分接，这样就可以形成单相回路，为单相负载供电
接在同一根火线上的所有单相用户为并联关系，电压都是220V，把他们看作一个整体，作为该相负载

## 220V和380V
![image.png](https://picture-01-1316374204.cos.ap-beijing.myqcloud.com/lenovo-xiangong/20260701114441304.png?imageSlim)
A相电压相量初相角为0°，B相滞后A相120°，所以在图中表示为顺时针旋转120°，同理C相电压相量超前A相电压120°，所以逆时针旋转120°。此时3个相电压对称。

星形接法中，线电压是等于两个相电压的相量差的，如图1-4所示，例如线电压$U_{AB}$ 等于相电压$U_A$ 减去$U_B$ 。注意，这里的相减不是简单的数值相减，而是相量的相减。至于为什么$U_{AB}$ 等于相电压$U_A$ 减去$U_B$ ，可以这样理解，中性点N接地，的电位为零，所以A相的电压其实就是端点A的电位，同理，端点B的电位为$U_B$ ，端点$A_B$间的电压为两点的电位差，这就是线电压$U_{AB}$ 。

## 家用插座
火线 + 零线 + 保护地线
1. 保护地线（PE 线）

- **位置**：三个插脚中最上方、通常更长更粗的那一个（品字形布局的顶部）
- **线色**：国标强制规定为**黄绿双色**，这是最容易识别的标志 —— 只要是黄绿相间的线，一定是地线
- **作用**：一端接电器的金属外壳，另一端接建筑的接地网。正常工作时没有电流，一旦电器内部漏电（火线碰到金属外壳），电流会通过地线直接导入大地，避免人体接触外壳时触电，属于安全保护的生命线。
- **设计细节**：地线插脚比火、零插脚更长，插拔时会「先接通地线、后断开地线」，全程保障安全。

2. 火线（L 线 / 相线）

- **位置**：下方两个短插脚之一
- **线色**：国标无强制唯一颜色，家用布线中常见**棕色、红色**
- **作用**：市电的供电进线，带有 220V 交流电压，和零线形成回路为电器供电。
- **特点**：对地存在 220V 电压，人体直接触碰会发生触电危险。

3. 零线（N 线 / 中性线）

- **位置**：下方另一个短插脚
- **线色**：家用布线中常见**蓝色、黑色**
- **作用**：电流的返回路径，电器工作时的电流从火线流入、零线流出，在小区变压器端零线是直接接地的。
- **特点**：正常情况下对地电压接近 0V，但工作时会有工作电流流过；三相负载严重不平衡时，零线可能带微弱电压。
