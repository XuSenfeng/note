# 四叉树

四叉树是 2D 平面上划分空间的一种数据结构，它可以更快速的回答「平面上一个位置附近有哪些对象」的问题， 常用于优化碰撞检测

## 分裂合并

首先，根节点管辖整张地图，然后，满足条件的情况下，将地图四等分，如果条件仍然满足，则继续下一层的四分，直到叶子节点管辖的区域不再可划分为止

![image.png](https://picture-01-1316374204.cos.ap-beijing.myqcloud.com/lenovo-xiangong/20260603182935173.png?imageSlim)


反过来，如果一个非叶子节点不再满足「可分裂」的条件了，也就是说，它原本的划分现在变的太细了，那么可以从叶子节点开始，向上合并回去。

比方说，移除一个对象之后，叶子节点空了，向上来看，叶子节点的父节点内也变地「空旷」了，那么，可以考虑把叶子节点和其兄弟叶子节点，一并释放掉，然后把它们剩余的管辖的对象还给父节点， 这个过程就是合并。
![image.png](https://picture-01-1316374204.cos.ap-beijing.myqcloud.com/lenovo-xiangong/20260603182942070.png?imageSlim)
## 区域查询

给定一个矩形区域，其左上角是 (x1,y1), 右下角是 (x2,y2)，如何查询出来它选中的所有对象呢

![image.png](https://picture-01-1316374204.cos.ap-beijing.myqcloud.com/lenovo-xiangong/20260603183527311.png?imageSlim)
思路也很简单，从根节点开始，不断「判重叠」。 如果当前节点管理的矩形 和 查询区域的矩形存在交叉重叠，那么：

1. 如果当前节点是**叶子节点**，直接收集位于查询区域范围内的对象就可以了。
2. 否则，如果当前是**非叶子节点**，那么向下继续判断其子节点，递归进行下去。非子叶比较大的一般是里面没有的, 比较小的可能存在

> 两个矩形是否相交可以使用[AABB检测平面相交](https://hit9.dev/post/aabb)
> `bool isOverlap = ax1 <= bx2 && ax2 >= bx1 && ay1 <= by2 && ay2 >= by1;

## 确定x的方格

在 $[0,N−1]$ 区间上的 N个整数，不断如此二分。第 d轮二分后，整数 x会落在 $2^dx/N$ 个小区间上

> 中点在不可以平分的时候取右边的点

第 d 轮的时候，均等划分了$2^d$ 个小区间，每个小区间的长度是 $k=N/2^d$，那么整数 x 就会落在第 x/k 个小区间上，也就是 $2^dx/N$

位置 (x,y) 一定位于 x 轴上第 $(2^d)x/h$ 个、y 轴上第 $(2^d)y/w$ 个小矩形内

![image.png](https://picture-01-1316374204.cos.ap-beijing.myqcloud.com/lenovo-xiangong/20260604091305838.png?imageSlim)

可以把整个区域划分为一个哈希表, 对于一个节点管辖区域里面的任意位置ID是一样的$(d, 2^dx/h, 2^dy/w)$
在查找一个位置 $(x,y)$ 所在的叶子节点时，关键在于确定深度 d 。因为 d 十分有限

## 查找邻居节点

输入的是一个节点 和 一个方向 (可选 8 个枚举值)，然后输出在这个方向上的邻居节点
在一个方向上的邻居并不一定只有一个叶子节点、也不一定是其兄弟节点
斜角方向是最容易处理的，因为在一个斜角方向上肯定只有一个邻居节点。
以左上角方向为例，当前节点管辖的小矩形的区域是 $(x_1,y_1)$ 和 $(x_2,y_2)$ ，那么，左上的相邻点 p 就是 $(x_1−1,y_1−1)$， 然后定位此位置归属的叶子节点即可
对于非斜角的方向，稍许复杂一点。这里面有两种情况， 有可能上面的邻居边长大于或者等于当前节点，也有可能上方是多个小边长的邻居
![image.png](https://picture-01-1316374204.cos.ap-beijing.myqcloud.com/lenovo-xiangong/20260604093645978.png?imageSlim)
都可以 先去找到上方边长不小于当前节点的邻居节点（它可能不是一个叶子节点），然后再向下递归寻找紧邻着当前节点的叶子节点作为答案
如果这个蓝色方框存在于一个大的叶子节点中，那么这个叶子节点就是唯一答案。
如果这个蓝色方框是一个中间节点，那么从此节点向下递归寻找「南」侧方向的子节点，直到叶子节点，就是所有答案。
