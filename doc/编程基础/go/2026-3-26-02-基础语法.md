## 导包

导的包里面的函数以及变量的首字母是大写的时候表示这个可以被其他的文件访问

```go
import "fmt"

	"fmt"
  "time"
)
```

导包的时候会执行包里面的init函数, 实际执行的时间是在兑入这个包的时候, 在main函数之前

```bash
import pkg1  --> 	 pkg1
	  ||	    <--			||
	const ... 	|		import pkg2 --> pkg2
	var ...			|			...
	init()			-- 	init()
	main()
		||
	 Exit
```

## defer

一个函数处理结束之前触发, 处理的方式是一个类似于压栈的处理形式, `return`执行比`defer`早

```go

import (
	"fmt"
)

func main() {
	defer fmt.Println("This is the end of main function")
	defer fmt.Println("This is the end of main function 2")

	fmt.Println("This is the main function")
}

/*

*/
```

## 类

一个类的首字母大写表示这个类可以被外部引用, 属性的首字母大写同理

```go
package main

import (
	"fmt"
)

type Person struct {
	Name string
	Age  int
}
// 为这个类创建方法
func (this Person) String() string {
	return fmt.Sprintf("Person{Name: %s, Age: %d}", this.Name, this.Age)
}
// 添加一个*表示这个类接收的是结构体的指针，方法内对字段的修改会影响原对象
func (this *Person) SetName(name string) {
	this.Name = name
}

func main() {
	p := Person{Name: "Alice", Age: 30}
	fmt.Println(p)

	p.SetName("Bob")
	fmt.Println(p)
}
```

### interface抽象类(多态)

interface（接口）是一种特殊的数据类型，用于定义一组方法的集合。它本身不存储数据，只规定实现了哪些方法的类型可以被赋值给该接口变量。

这个interface里面只可以有方法, 没有实际的属性

```bash
package main

import (
	"fmt"
)

type Person interface {
	SetName(name string)
	String() string
}

type Superman struct {
	Person
	Name  string
	Age   int
	Speed int
}

type Batman struct {
	Person
	Name  string
	Age   int
	Speed int
}

func (this *Superman) SetName(name string) {
	this.Name = name
}

func (this Superman) String() string {
	return fmt.Sprintf("Superman{Name: %s, Age: %d, Speed: %d}", this.Name, this.Age, this.Speed)
}

func (this *Batman) SetName(name string) {
	this.Name = name
}

func (this Batman) String() string {
	return fmt.Sprintf("Batman{Name: %s, Age: %d, Speed: %d}", this.Name, this.Age, this.Speed)
}

func PrintName(p Person) {
	fmt.Println(p.String())
}

func main() {
	s := Superman{Age: 30, Speed: 100, Name: "jiao"}
	PrintName(&s)

	b := Batman{Age: 35, Speed: 50, Name: "bruce"}
	PrintName(&b)
}
```

### 万能类型

interface{} 在 Go 语言中被称为“空接口”，它可以表示任何类型的值。因为所有类型都实现了空接口，所以 interface{} 可以用来存储任意类型的数据，相当于“万能类型”或“通用容器”, 可以用于变参数的函数的传递数值

```go
var a interface{}
a = 123
a = "hello"
a = []int{1, 2, 3}
```

```go
package main

import (
	"fmt"
)

func my_Func(arg interface{}) {
	switch v := arg.(type) {
	case int:
		fmt.Printf("Argument is an integer: %d\n", v)
	case string:
		fmt.Printf("Argument is a string: %s\n", v)
	default:
		fmt.Printf("Argument is of unknown type: %T\n", v)
	}
}

func main() {
	my_Func(42)
	my_Func("Hello, World!")
	my_Func(3.14)
}
```

可以使用类型断言进行判断实际的类型

```go
value, ok := arg.(string)
```



## 断言

类型断言（type assertion）是 Go 语言中用于将接口类型的变量转换为具体类型或另一个接口类型的语法。

语法：

-   x 是接口类型变量
-   T 是你想要断言的目标类型

作用：

-   判断接口变量 x 的实际类型是否为 T，如果是，则转换成功，否则会 panic。

```go
value, ok := x.(T)
if ok {
    // 转换成功，可以安全使用 value
} else {
    // 转换失败
}
```

go里面的变量分为两部分, type以及value, 其中的type可以划分为两种, `static type`以及`concrete type`二选一

+   static type: 是一个`int、string...`类型的数据
+   `concrete type`: 是一个interface指向的类型, 系统看得见的类型

实际使用的一个类可能实现不同的interface里面的函数, 所以可以进行类型的转换

```go
package main

import (
	"fmt"
)

type Reader interface {
	ReadBook()
}

type Writer interface {
	WriteBook()
}

type Book struct {
	Title string
}

func (b Book) ReadBook() {
	fmt.Printf("Reading book: %s\n", b.Title)
}

func (b Book) WriteBook() {
	fmt.Printf("Writing book: %s\n", b.Title)
}

func main() {
	book := Book{Title: "The Great Gatsby"}
	var r Reader = book
	var w Writer = r.(Writer)

	r.ReadBook()
	w.WriteBook()
}
```

## 反射

动态的获取一个变量的type以及value

```go
func reflectType(arg interface{}) {
	fmt.Println("type: ", reflect.TypeOf(arg))
	fmt.Println("value: ", reflect.ValueOf(arg))
}

var num int = 42
reflectType(num)
```

也可以处理比较复杂的类型

```go
package main

import (
	"fmt"
	"reflect"
)

type User struct {
	Name string
	Age  int
}

func (this User) Call() {
	fmt.Printf("User Name: %s, Age: %d\n", this.Name, this.Age)
}

func main() {

	user := User{Name: "Alice", Age: 30}
	inputType := reflect.TypeOf(user)
	fmt.Println("inputType: ", inputType.Name())

	inputValue := reflect.ValueOf(user)
	fmt.Println("inputValue: ", inputValue)

	// 通过type获取里面的字段
	for i := 0; i < inputType.NumField(); i++ {
		field := inputType.Field(i)
		value := inputValue.Field(i).Interface()
		fmt.Printf("Field: %s, Type: %s, Value: %v\n", field.Name, field.Type, value)
	}

	// 通过type获取里面的方法
	for i := 0; i < inputType.NumMethod(); i++ {
		method := inputType.Method(i)
		fmt.Printf("Method: %s, Type: %s\n", method.Name, method.Type)
	}
}
```

## Tag标签

可以给struct里面的元素打一个标签, 进一步描述

```go
package main

import (
	"fmt"
	"reflect"
)

type resume struct {
	Name string `json:"name" validate:"required"`
	Age  int    `json:"age"`
}

func FindTag(str interface{}) {
	t := reflect.TypeOf(str).Elem()
	for i := 0; i < t.NumField(); i++ {
		tagstring := t.Field(i).Tag.Get("json")
		fmt.Printf("Tag: %s\n", tagstring)
	}
}

func main() {
	r := resume{Name: "jiao", Age: 30}
	FindTag(&r)
}
```

### json

这个tag一般使用在json生成

```go
package main

import (
	"encoding/json"
)

type Persion struct {
	Name string `json:"name"`
	Age  int    `json:"age"`
}

func main() {
	persion := Persion{
		Name: "张三",
		Age:  18,
	}
	data, err := json.Marshal(persion) // 将结构体转换为JSON格式的字节切片
	if err != nil {
		panic(err)
	}
	println(string(data))
	var persion2 Persion
	err = json.Unmarshal(data, &persion2) // 将JSON格式的字节切片转换回结构体
	if err != nil {
		panic(err)
	}
	println(persion2.Name)
	println(persion2.Age)
}
```
