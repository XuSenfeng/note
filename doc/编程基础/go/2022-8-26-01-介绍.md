# Go

Go 语言被设计成一门应用于搭载 Web 服务器，存储集群或类似用途的巨型中央服务器的系统编程语言。

对于高性能分布式系统领域而言，Go 语言无疑比大多数其它语言有着更高的开发效率。它提供了海量并行的支持

## 特点

+   部署简单, 编译为机器码
+   不依赖其他的库
+   可以直接运行
+   静态类型语言, 编译的时候可以检查出来大部分问题
+   天生的编发, 可以支持多核, 底层自动优化, runtime系统调度机制
+   高效的垃圾回收
+   丰富的标准库

## 运行编译

```go
package main // 当前文件的包名字, main表示是主要的包

import "fmt"

func main() {
    fmt.Println("Hello, World!")
}
```

```bash
$ go run hello.go 
Hello, World!
# 编译
$ go build hello.go 
$ ls
hello    hello.go
$ ./hello 
Hello, World!
```
