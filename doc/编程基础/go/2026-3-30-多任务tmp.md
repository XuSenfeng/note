# 多任务



## channel

```go
package main

import (
	"fmt"
)

func main() {
	ch := make(chan int)

	go func() {
		for i := 0; i < 5; i++ {
			ch <- i
		}
		close(ch)
	}()

	for {
		if num, ok := <-ch; ok {
			fmt.Println(num)
		} else {
			break
		}
	}
}
```

当确定没有再进行数据写入的时候, 可以关闭它, 关闭以后不可以再收发数据, 在接收的部分会收到一个`ok=false`(获取所有数据以后)

>   没有经过make初始化的channel收发数据的时候会发生阻塞

### 关键字配合

+   for循环

```c
for data := range ch {
	// 处理

```

+   select处理不同的channel

```go
package main

import (
	"fmt"
)

func main() {
	c := make(chan int)
	quit := make(chan bool)
	
	go func() {
		for i := 0; i < 5; i++ {
			c <- i
		}
		quit <- true
	}()
	
	for {
		select {
		case v := <-c: // 接收c可读信号，打印接收到的值
			fmt.Printf("Received: %d\n", v)
		case <-quit: // 接收quit可读信号，退出循环
			fmt.Println("Quitting...")
			return
		}
	}
}
```
