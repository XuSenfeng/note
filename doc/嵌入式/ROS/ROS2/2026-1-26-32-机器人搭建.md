## 软件

安装platform IO

使用Vscode插件进行安装, 安装以后的插件在用户目录里面创建`.platformio`文件夹, 可以在`penv`子目录里面`. bin/activate`进行激活, `pip install platformio`

`pio project init`: 创建项目
