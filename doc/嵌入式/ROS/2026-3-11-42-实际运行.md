## 参数

机器人有两个地图, “全局代价地图”、“局部代价地图”, 使用这两关地推去计算实际的路径

1.  全局代价地图（`global_costmap`）

-   基于静态地图 `/map`（再叠加部分障碍信息）
-   全局规划器在这张图上算一条从起点到终点的“长路径”（粗路线）

2.   局部代价地图（`local_costmap`）

-   机器人周围滚动窗口，实时融合激光障碍
-   局部规划器在这张图上做“短时可执行轨迹”，负责绕开临时障碍、跟踪全局路径

3.   实际运动控制

-   局部规划器输出 `/cmd_vel`（线速度/角速度）
-   底盘驱动节点接收 `/cmd_vel`，控制电机运动

### 导航

参数的位置在`limo_bringup/param/diff`路径下面

+   `costmap_common_params.yaml`: 通用参数, 记录小车的大小
+   `global_costmap_params.yaml`: 全局地图的配置文件
+   `local_costmap_params.yaml`: 局部地图的配置文件
+   `planner.yaml`: 规划器的配置参数

## 深度相机

安装(新车没有)

```bash
sudo apt install ros-melodic-realsense2-camera ros-melodic-realsense2-description

sudo apt-key adv --keyserver keyserver.ubuntu.com --recv-key F6E65AC044F831AC80A06380C8B3A55A6F3FCBBAB
sudo add-apt-repository "deb https://librealsense.intel.com/Debian/apt-repo bionic main" -u

# 2. 安装完整驱动（含 viewer 工具）
sudo apt install -y librealsense2-dkms librealsense2-utils librealsense2-dev librealsense2-dbg

# 3. 验证安装（查看版本+确认 viewer 路径）
pkg-config --modversion librealsense2  # 输出版本号（如 2.50.0）则正常
which realsense-viewer  # 输出 /usr/bin/realsense-viewer 则工具安装成功
```
