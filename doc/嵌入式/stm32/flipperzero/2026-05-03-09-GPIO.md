# GPIO

## 可用gpio

```c
{.pin = &gpio_ext_pa7,      // 底层硬件引脚对象
 .name = "PA7",             // 芯片上的引脚名(MCU 的端口名)
 .channel = FuriHalAdcChannel12,  // 能做 ADC(模拟量读取)的通道
 .pwm_output = ...,         // 能输出 PWM 的话,对应哪个定时器通道
 .number = 2,               // 排针上的物理位置编号(第几号针脚)
 .debug = false},           // 是否为调试/危险引脚(需保护)

```

所有的 GPIO 使用数组 `const GpioPinRecord gpio_pins[]` 进行记录

```
1  → 5V         (电源,不在数组里)
2  → PA7
3  → PA6
4  → PA4
5  → PB3
6  → PB2
7  → PC3
8  → GND        (地,不在数组里)
9  → 3V3        (电源)
10 → PA14 (SWCLK)   ← .debug=true
11 → GND
12 → PA13 (SWDIO)   ← .debug=true
13 → PB6  (USART_TX)
14 → PB7  (USART_RX)
15 → PC1
16 → PC0
17 → PB14 (iButton)
18 → GND
```

除了顶部的排针还有两个可以使用的引脚

```c
gpio_speaker (PB8)、gpio_infrared_tx (PB9)
```

> 注释标注, 使用可能导致硬件损毁


## 基础使用

```c
#include "furi_hal.h"

int gpio_control_main(void* p) {
    UNUSED(p);

    furi_hal_gpio_init_simple(&gpio_ext_pa7, GpioModeOutputPushPull);
    do {
        furi_hal_gpio_write(&gpio_ext_pa7, true);
        furi_delay_ms(500);
        furi_hal_gpio_write(&gpio_ext_pa7, false);
        furi_delay_ms(500);
    } while(furi_hal_gpio_read(&gpio_button_ok));
    furi_hal_gpio_init_simple(&gpio_ext_pa7, GpioModeAnalog);

    return 0;
}

```

## PWM

```bash
// 在PA7引脚设置一个10Hz的50占空比的PWM波形
furi_hal_pwm_start(FuriHalPwmOutputIdTim1PA7, 10, 50);
do {
    furi_delay_ms(500);
} while(furi_hal_gpio_read(&gpio_button_ok));
furi_hal_pwm_stop(FuriHalPwmOutputIdTim1PA7);
```
