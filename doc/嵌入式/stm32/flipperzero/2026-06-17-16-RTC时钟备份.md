
# RTC

**RTC = Real-Time Clock(实时时钟)**,是 STM32 芯片**内部独立的一个外设模块**,不是软件结构,而是硅片上的一块专用电路。它的核心物理特性是:

- **有自己独立的时钟源**(LSE 32.768 kHz 晶振 / LSI 内部低速振荡器);
- **有自己独立的供电域**("Backup Domain"),由备份电池供电;
- **主芯片掉电、复位、深度睡眠都不影响它**——只要纽扣电池还有电,RTC 就一直在跑。

但实际的Flipperzero使用的RTC还有其他的功能, 可以用于掉电不丢失的备份
## 备份
flipperzero的RTC处理基础的RTC功能以外, 还有一组**Backup Registers(备份寄存器)**, 32 个 32-bit 的 Backup Registers
