# GUI模块

## 组件Widget

Widget 是 Flipper Zero UI 库中一个非常通用的**多功能容器视图**。

你可以根据需要在上面排放各种预定义的 UI 元素（如文字、图标、按钮等）。当你不想为一个简单的显示页面专门写一个复杂的绘图回调函数（Canvas callback）时，Widget 是最方便的选择

这些小部件包括 `Button` 、 `Icon` 、 `String` 、 `String (Multi-Line)` 等等

```c
// 申请
Widget* widget_alloc(void);

// 释放
void widget_free(Widget* widget);

// 清空
void widget_reset(Widget* widget);

// 获取View
View* widget_get_view(Widget* widget);

// 一个支持换行符的字符串类型
void widget_add_string_multiline_element(
    Widget* widget,
    uint8_t x,
    uint8_t y,
    Align horizontal,
    Align vertical,
    Font font,
    const char* text);

// 添加一个字符显示的组件
void widget_add_string_element(
    Widget* widget,
    uint8_t x,
    uint8_t y,
    Align horizontal,
    Align vertical,
    Font font,
    const char* text);

/** 固定大小的文本框：文本会在给定的 width/height 区域内排版显示
	超出时不会滚动，而是可选地用 ... 截断
 * @param[in]  text             可以使用的格式
     "\e#Bold text\e#" - 加粗
     "\e*Monospaced text\e*" - 等宽
     "\e!Inversed text\e!" - 白字黑背景
 * @param      strip_to_dots    是否截断
 */
void widget_add_text_box_element(
    Widget* widget,
    uint8_t x,
    uint8_t y,
    uint8_t width,
    uint8_t height,
    Align horizontal,
    Align vertical,
    const char* text,
    bool strip_to_dots);

/* 滚动文本显示
 The following formats are available:
 "\e#Bold text" - 设置字体加粗
 "\e*Monospaced text\e*" 设置字体的等宽
 "\ecCenter-aligned text" 设置字体居中
 "\erRight-aligned text" 设置字体右对齐
 */
void widget_add_text_scroll_element(
    Widget* widget,
    uint8_t x,
    uint8_t y,
    uint8_t width,
    uint8_t height,
    const char* text);

// 绘制一个按钮
void widget_add_button_element(
    Widget* widget,
    GuiButtonType button_type,
    const char* text,
    ButtonCallback callback,
    void* context);

// 绘制一个Icon
void widget_add_icon_element(Widget* widget, uint8_t x, uint8_t y, const Icon* icon);

// 画一个框
#define widget_add_frame_element(widget, x, y, width, height, radius) \
    widget_add_rect_element((widget), (x), (y), (width), (height), (radius), false)

// 画长方形
void widget_add_rect_element(
    Widget* widget,
    uint8_t x,
    uint8_t y,
    uint8_t width,
    uint8_t height,
    uint8_t radius,
    bool fill);

// 画圆
void widget_add_circle_element(Widget* widget, uint8_t x, uint8_t y, uint8_t radius, bool fill);

// 画线
void widget_add_line_element(Widget* widget, uint8_t x1, uint8_t y1, uint8_t x2, uint8_t y2);
```
