# 串口

```bash
sudo stty -F /dev/ttyS7 115200 cs8 -cstopb -parenb -ixon -ixoff -crtscts raw -echo
sudo cat /dev/ttyS7 | xxd          # 用 xxd 把二进制变 hex 显示，纯文本可以省掉

echo -n "hello" | sudo tee /dev/ttyS7 > /dev/null
```

- `115200` 波特率（按你设备实际改 9600/115200/...）
- `cs8` 8 位数据
- `-cstopb` 1 位停止位（加 `cstopb` 是 2 位）
- `-parenb` 无校验（`parenb` + `-parodd` = 偶校验，`parenb parodd` = 奇校验）
- `-ixon -ixoff` 关流控
- `-crtscts` 关硬件流控
- `raw -echo` 关行处理 / 关回显，必加，否则数据会被改
