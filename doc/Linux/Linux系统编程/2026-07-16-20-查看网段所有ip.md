
# linux

`arp -a` 查看arp缓存表, 里面记录通信过的ip以及mac地址
可以使用命令`arp -i eth0`指定网口

ARP 缓存是被动积累的：只有当你的机器和某个 IP 通信过（发过包、ping 过等），才会触发 ARP 请求并把对方的 IP+MAC 记录下来。没通信过的设备，缓存里就没有

```bash
# ping 网段广播地址（触发所有主机回应） 
ping -b 192.168.1.255 -c 3 # 或者用 for 循环逐个 ping 
for i in $(seq 1 254); do ping -c 1 -W 0.2 192.168.1.$i & done; wait # 然后查看结果 
arp -n
```

可以使用专门的工具

```bash
sudo arp-scan -l # 扫描本地子网 
sudo arp-scan 192.168.1.0/24 # 指定网段 
sudo arp-scan -I eth0 192.168.1.0/24 # 指定网卡
```
