# 产品设计

## 字符

从一个字符的编号到这个字符的图像, 需要使用字符的点阵, 但是点阵的大小是不可以变化的, 这个时候可以使用freetype, Freetype是开源的字体引擎库, 使用的是一个矢量字体

实现的原理是通过去取出关键点, 把点连起来, 闭合曲线填充颜色

FreeType**本身不直接处理 UTF-8、GBK、GB2312 等字节编码**，它的核心是基于**Unicode 码点（Code Point）** 来索引字体中的字形（Glyph）。

> Unicode 是**字符的唯一数字标识（码点）**，解决 “全球字符统一编号” 的问题；字节编码是**码点转字节的规则**，解决 “存储 / 传输” 的问题
