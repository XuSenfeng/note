## 打开

### 文件初始化

+ 初始化inode节点

在函数`init_special_inode`里面, 会配置这个文件的`i_fop`是`def_chr_fops`, 这根个结构体里面配置有

```c
const struct file_operations def_chr_fops = {
	.open = chrdev_open,
	.llseek = noop_llseek,
};
```

+ cdev_add

字符设备和查找表的映射, 记录设备的设备号, 总结父设备的引用次数

### openc函数

在打开的时候会使用`chrdev_open`函数, 这个函数查找替换真实的对应的cdev的open函数
