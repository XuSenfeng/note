# 入门

## 下载

实际安装的时候需要安装两个

1. EMMC里面的基础系统
2. SD卡里面的完整系统

EMMC的系统使用Ubuntu里面安装的Nvidia Jetpack(https://developer.nvidia.com/sdk-manager）

```bash
sudo dpkg -i sdkmanager_1.5.0-7774_amd64.deb
```

之后在SDK里面进行安装, 开发板的引脚短接
