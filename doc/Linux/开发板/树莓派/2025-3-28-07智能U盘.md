### 软件

+ 文件辅助控制

#### 文件共享

使用网络进行共享工作区的文件, 类似U盘形式

使用USB接口模拟键盘, 通过键盘快捷键的触发电脑功能(开启对应的软件/磁盘挂载)

## 实际实现

```bash
sudo apt-get install vim
```

### 文件分享

```bash
sudo apt-get install samba
```

在文件的/etc/samba/smb.conf添加

```
[myshare]
# 可匿名访问
path = /home/jiao/.picoclaw/workspace
writeable=Yes
create mask=0777
directory mask=0777
public=yes
browseable=yes
```

```bash
sudo smbpasswd -a pi
```

> 使用BCM码标记

|   功能    | 对应按键 |     描述     |
| :-------: | :------: | :----------: |
|   KEY1    |   P21    |  屏幕按键1   |
|   KEY2    |   P20    |  屏幕按键2   |
|   KEY3    |   P16    |  屏幕按键3   |
|  摇杆up   |    P6    |              |
| 摇杆Down  |   P19    |              |
| 摇杆Left  |    P5    |              |
| 摇杆Right |   P26    |              |
| 摇杆Press |   P13    |              |
|   SCLK    |   P11    |   屏幕通信   |
|   MOSI    |   P10    |              |
|    DC     |   P25    |              |
|    CS     |  P8/CES  |              |
|    RST    |   P27    |              |
|    BL     |   P24    |              |
|    SDA    |    P2    | 电源模块通信 |
|    SCL    |    P1    |   电源模块   |
| 充电检测  |    P4    | 电源是否插入 |
|           |          |              |
|           |          |              |
