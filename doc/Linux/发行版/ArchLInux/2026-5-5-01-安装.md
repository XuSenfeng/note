# 安装

## 下载镜像

https://mirrors.tuna.tsinghua.edu.cn/archlinux/iso/2026.05.01/

之后创建一个新的虚拟机镜像, 启动即可

## 安装

### ssh连接

```bash
ip addr show # 查看IP
systemctl restart sshd # 重启ssh
passwd
# 之后使用ssh连接
```

### 换源

可以在https://archlinux.org/mirrorlist/查看, 生成使用的配置文件, 选择China, 不使用http

```bash
pacman -Sy # 更新
pacman-key --init # 初始化 Pacman 密钥环（创建密钥存储文件）
pacman-key --populate archlinux
pacman -S wget
wget "https://archlinux.org/mirrorlist/?country=CN&protocol=https&ip_version=4" --output-document=/etc/pacman.d/mirrorlist
# 使用Vim在这个文件里面启用实际使用的源
```

### 安装

```bash
archinstall
```

## 命令行

### 初始化分区

```bash
mkfs.fat -F 32 /dev/nvme1n1p1 # 格式化第一个分区, 这个分区是启动分区, 使用fat32
mkfs.btrfs /dev/nvme1n1p2 # 格式化第二个分区

cfdisk # 分区, New, 1g, Type->EFI System, 剩下的分到一个分区, 使用Write->yes, 退出
lsblk -pf

# 格式化分区
mount -t btrfs -o compress=zstd /dev/nvme1n1p2 /mnt
btrfs subvolume create /mnt/@
btrfs subvolume create /mnt/@home
btrfs subvolume create /mnt/@swap
unmount /mnt
mount -t btrfs -o subvol=/@,compress=zstd /dev/nvme1n1p2 /mnt
mount --mkdir -t btrfs -o subvol=/@home,compress=zstd /dev/nvme1n1p2 /mnt/home
mount --mkdir -t btrfs -o subvol=/@swap,compress=zstd /dev/nvme1n1p2 /mnt/swap
mount --mkdir /dev/nvme1n1p1 /mnt/boot
mount --mkdir /dev/nvme0n1p1 /mnt/winboot
# 查看实际的情况
df -h


```
