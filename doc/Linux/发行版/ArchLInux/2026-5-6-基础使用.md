# 基础使用
## 备份
使用以下的命令创建快照
```bash
sudo snapper  -c home create --description "完成界面"
sudo snapper  -c root create --description "完成界面"
```
创建以后可以使用命令进行查看
```bash
sudo snapper  -c home list
```
如果是home分区， 可以直接回滚
```bash
sudo snapper -c home undochange 10..0
```
根分区正在被系统使用，**直接在线回滚会导致系统崩溃、无法开机**
```bash
sudo snapper -c root rollback 10 -d "回滚到完成界面"
```
此命令会自动创建当前状态的备份快照，再设置目标快照为**默认子卷**
之后重启

## 安装软件

可以直接下载AppImage格式的软件进行安装

**AppImage 是 Linux 专属的「绿色免安装便携软件格式」**，就像 Windows 里的 **便携版单文件绿色软件**，**一个文件 = 整个软件 + 所有依赖库 + 运行环境**。

### 放入图标

```bash
# 把 AppImage 移到固定目录
mv ~/下载/imager_2.0.7_amd64.AppImage ~/.local/bin/

# 加可执行权限
chmod +x ~/.local/bin/imager_2.0.7_amd64.AppImage
```

写一键启动脚本（自动处理 xhost + sudo）

```bash
vim ~/.local/bin/pi-imager-run
```


```bash
#!/bin/bash
# 允许 root 访问图形界面
xhost +si:localuser:root
# 强制 XCB 平台、sudo 启动镜像工具
sudo /home/jiao/.local/bin/imager_2.0.7_amd64.AppImage
```

保存退出，给脚本加执行权限：

```bash
chmod +x ~/.local/bin/pi-imager-run
```

sudo 免密（点开图标不用输密码）

```bash
sudo visudo
```

拉到文件**最后一行**，添加：

```plaintext
jiao ALL=(ALL) NOPASSWD: /home/jiao/.local/bin/imager_2.0.7_amd64.AppImage
```

保存退出：`Ctrl+O` 回车，`Ctrl+X`

创建桌面图标文件（应用菜单可搜到）

```bash
vim ~/.local/share/applications/raspberry-imager.desktop
```

完整复制下面内容（路径已经帮你写好，不用改）：

```ini
[Desktop Entry]
Type=Application
Name=Raspberry Pi Imager
Comment=树莓派系统镜像烧录工具
Exec=/home/jiao/.local/bin/pi-imager-run
Icon=drive-removable-media
Terminal=false
Categories=System;Utility;
StartupWMClass=imager
```

刷新系统应用缓存

```bash
update-desktop-database ~/.local/share/applications/
```

## Pacman使用


+ 安装
```bash
pacman -S _包名_1_ _包名_2_
```

+ 包组
有的包组包含大量的软件包，有时用户只需其中几个
```bash
Enter a selection (default=all): 1-10 15
```
可以使用`^`进行取反
使用命令可以查看这个组里面实际有哪些包
```bash
pacman -Sg gnome
```

+ 删除
```bash
pacman -R _package_name_
# 删除指定软件包，及其所有没有被其他已安装软件包使用的依赖关系
pacman -Rs _package_name_
```
+ 升级
升级整个系统
```bash
pacman -Syu
```
+ 查看一个包的依赖
```bash
pactree _package_name_
```
