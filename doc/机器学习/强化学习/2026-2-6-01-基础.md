# 基础

## 基础概念

### grid-world example

有一个网格世界, 里面有一个机器人, 网格有几种, Accessible/forbidden/target/boundary

尝试查找一个最优的路径以及减少不必要的拐弯

![image-20260206104807680](https://picture-01-1316374204.cos.ap-beijing.myqcloud.com/lenovo-picture/202602061048741.png)

### State状态

当前的agent相对于环境的状态, 在网格世界里面, 不同的位置对应不同的状态, `s1, s2, s3 ... s9`

所有的状态放在一起是状态空间, State Space, $S = \{s_{i}\}_{i=1}^{9}$

### Action行为

在每一个状态可以采取的行动, `a1, a2, a3`

放在一起的时候是状态的集合$A(s_{i}) = \{a_{i}\}_{i=1}^{5}$

### State transition

一个状态使用一个action以后可以变为另一个state, $s1\xrightarrow{a_1}s2$, 定义的是交互的行为

可以使用表格的形式进行展示, 用于展示确定性的变化

![image-20260206105907218](https://picture-01-1316374204.cos.ap-beijing.myqcloud.com/lenovo-picture/202602061059270.png)

更经常使用的是概率$p(s_{2}|s_{1}, a_{2}) = 1$, 从s1到s2的概率

### Policy策略

在某一个状态应该使用的行为

![image-20260206110312598](https://picture-01-1316374204.cos.ap-beijing.myqcloud.com/lenovo-picture/202602061103644.png)

实际使用的还是条件概率, 使用$\pi$表示一个策略$\pi(a_{1}|s_{1}) = 1$, 一个状态的所有策略的概率和是1, 可以是不确定的

### Reward

正数表述鼓励, 负数是惩罚, 0的时候通常是表示鼓励, 用于引导, $p(r = -1 |s_{1}, a_{1}) = 1$, $p(r \neq -1 |s_{1}, a_{1}) = 0$, 表示在s1的时候使用a1的结果

### Trajectory轨迹和return

$s1\xrightarrow[r=0]{a_1}s2\xrightarrow[r=0]{a_3}s5\xrightarrow[r=1]{a_2}s9$

一系列的运动状态和reward的集合, $return = 0 + 0 + 1 = 1$

### Discounted return

$s1\xrightarrow[r=0]{a_1}s2\xrightarrow[r=0]{a_3}s5\xrightarrow[r=1]{a_2}s9\xrightarrow[r=1]{a_2}s9\xrightarrow[r=1]{a_2}s9\xrightarrow[r=1]{a_2}s9\xrightarrow[r=1]{a_2}s9$

到达目标以后一直可以获取到1的数值, 这时候可以引入一个$\gamma \in [0, 1] $进行return的衰减

$return = 0 +\gamma0 +\gamma^{2}1 +\gamma^{3}1 +\gamma^{4}1$

可以通过控制这个值达到使得机器更加重视比较近的值还是比较远的值

### Episode章节

一个有限步数的Trajectory, 到达一定的状态`terminal state`以后, 停止, 有的任务是没有终止状态的`continuing task`, 一般会把episode任务转换为continuing task

+ 设置target state为一个`absorbing state`到达目标状态已以后, 不论采取什么都会回到这个状态, 但是获取的reward为0
+ 设置这个状态是一个普通的状态, agent可以离开这里再次进入

### Markov decision process(MDP)

是一个框架

+ Sets:
+ - State space: the set of all states, denoted as S.
    - Action space: a set of actions, denoted as $A(s)$, associated with each state s ∈ S. 
    - Reward set: a set of rewards, denoted as $R(s,a)$, associated with each state-action pair (s,a). 
+ Probability distruction
+ + $p(s'|s, a)$采取不同行动的到达另一个位置的概率
    + $p(r|s, a)$使用一个行动可以获取的回报
+ Policy
+ + $\pi(a|s)$在一个位置使用不同策略的概略
+ Markov property: 和记忆无关的
+ + $p(s_{t+1}|a_{t+1},st, ...,  a1, s_{0}) = p(s_{t+1}|a_{t+1}, s_{t})$, 采取行动的概率和之前无关
    + $p(r_{t+1}|a_{t+1},st, ...,  a1, s_{0}) = p(r_{t+1}|a_{t+1}, s_{t})$

once the policy in an MDP is fixed, the MDP degenerates into an MP 规则固定的时候, 变为Markov process
