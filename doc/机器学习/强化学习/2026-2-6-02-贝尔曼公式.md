# 贝尔曼公式

## State value

### return

在同一个位置的时候, 选择不同的路线可以获取到不同的return值, 不同位置的选择可以是不确定的

![image-20260206123948262](https://picture-01-1316374204.cos.ap-beijing.myqcloud.com/lenovo-picture/202602061239309.png)

$return_{1} =0 + \gamma + \gamma^{2} + ... =  \frac{\gamma}{1-\gamma}$

$return_{2} =-1 + \frac{\gamma}{1-\gamma}$

$return_{2} =0.5 + \frac{\gamma}{1-\gamma}$

用于评估不同的策略

#### 计算return

![image-20260206124345399](https://picture-01-1316374204.cos.ap-beijing.myqcloud.com/lenovo-picture/202602061243438.png)

$v_{1} = r_{1} + \gamma r_{2} +\gamma^{2}r_{3} + ...$

$v_{2} = r_{2} + \gamma r_{3} +\gamma^{2}r_{4} + ...$

可以推导出来$v_{1} = r_{1} + \gamma v2$, 这是一个Bootstrapping

> 源自英文谚语**pull oneself up by one's bootstraps**（靠靴带把自己拉起来），本义是 “凭自身力量完成看似不可能的事”，中文常译**自举**

![image-20260206125256442](https://picture-01-1316374204.cos.ap-beijing.myqcloud.com/lenovo-picture/202602061252493.png)

也可以写成$v = r + \gamma Pv$, 这就是贝尔曼公式的简单示例, $v = (I - \gamma P )^{-1}r$, 因此之前的那个实例可以表示为

+ $v_{1} = r + \gamma v_{3}$
+ $v_{2} = r + \gamma v_{4}$
+ $v_{3} = r + \gamma v_{4}$
+ $v_{4} = r + \gamma v_{4}$

### State Value

$S_{t}\xrightarrow{A_{t}}R_{t+1}, S_{t+1}$

这里面的大写代表是一个随机的数值, 可以使用概率分布(probabity distributions)进行计算

> $\pi (A_{t} = a|S_{t} = s)$, $p(R_{t+1} = r|S_{t} = a, A_{t} = a)$, $p(S_{t+1} = s'|S_{t} = a, A_{t} = a)$

可以进一步推导出来多步的变化

$G_{t} = R_{t+1} + \gamma R_{t+2} + ...$, G~t~也是一个随机变量(radom variables), 可以计算期望
$$
v_{\pi}(s) =  \mathbb{E}[G_{t}|S_{t} = s]
$$
使用不同的策略$\pi$可以获取到不同的State Value, 不同状态的获取到的数值也是不同的

## Bellman equation贝尔曼公式

描述不同状态的State Value 之间的关系
$$
G_{t} = R_{t+1} + \gamma G_{t+1}
$$

$$
v_{\pi}(s) =\mathbb{E}[R_{t+1} + \gamma G_{t+1}|S_{t} = s]=\mathbb{E}[R_{t+1}|S_{t} = s] + \gamma \mathbb{E}[G_{t+1}|S_{t}=s]
$$

## Action Value

State Value: 从一个状态出发得到的return的平均值

Action Value: 从一个状态出发, 选择一个action以后得获得的值
$$
q_{\pi}(s, a) = \mathbb{E}[G_{t}|S_{t} = s, A_{t} = a]
$$
