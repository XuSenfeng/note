# 值迭代/策略迭代

## 值迭代

$$
v_{k+1} = f(v_{k}) = max _{\pi}(r_{\pi} + \gamma P_{\pi}v_{k}), k = 0, 1, 2
$$

可以使用类似于这种的迭代的方式获取实际的数值, 这个方式叫做值迭代

在每一次迭代的时候有两个步骤

+ $\pi_{k+1} = argmax_{\pi}(r_{\pi} + \gamma P_{\pi}v_{k})$, policy update, 获取实际使用的策略, 策略迭代, 获取到当前的值可以获取的最优策略
+ $v_{k+1} = r_{\pi_{k+1}} + \gamma P_{\pi_{k+1}}v_{k}$ value update, 使用这个策略更新一下数值

## 策略迭代

也是分为两个步骤

+ policy evaluation(PE): 计算一下当前$\pi_{k}$的state value, $v_{\pi k} = r_{\pi_{k}}+ \gamma P_{\pi k}v_{\pi k}$
+ policy improvement(PI): $\pi_{k+1} = argmax_{\pi}(r_{\pi} + \gamma P_{\pi}v_{\pi_{k}})$优化一下策略, 这一步可以获取更大的$v_{\pi k}$

## 比较

从一个$v_{0}$出发, 先获取一个$\pi_{0}$

+ $\pi_{k+1} = argmax_{\pi}(r_{\pi} + \gamma P_{\pi}v_{k})$, policy update, 获取实际使用的策略, 策略迭代, 获取到当前的值可以获取的最优策略
+ $v_{k+1} = r_{\pi_{k+1}} + \gamma P_{\pi_{k+1}}v_{k}$ value update, 使用这个策略更新一下数值

从一个$\pi_{0}$出发

+ policy evaluation(PE): 计算一下当前$\pi_{k}$的state value, $v_{\pi k} = r_{\pi_{k}}+ \gamma P_{\pi k}v_{\pi k}$
+ policy improvement(PI): $\pi_{k+1} = argmax_{\pi}(r_{\pi} + \gamma P_{\pi}v_{\pi_{k}})$优化一下策略, 这一步可以获取更大的$v_{\pi k}$
