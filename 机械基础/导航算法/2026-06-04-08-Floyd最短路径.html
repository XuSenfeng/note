Floyd最短路径

实现逻辑

弗洛伊德算法定义了两个二维矩阵：

  1. 矩阵D记录顶点间的最小路径

例如D[0][3]= 10，说明顶点0 到 3 的最短路径为10；
2. 矩阵P记录顶点间最小路径中的中转点
例如P[0][3]= 1 说明，0 到 3的最短路径轨迹为：0 -> 1 -> 3。
它通过3重循环，k为中转点，v为起点，w为终点，循环比较D[v][w] 和 D[v][k] + D[k][w] 最小值，如果D[v][k] + D[k][w] 为更小值，则把D[v][k] + D[k][w] 覆盖保存在D[v][w]中

代码实现

#include <stdio.h>
#include <stdlib.h>

#define MAXN 10 
#define INF 1000

typedef struct struct_graph{
    int vexnum;//顶点数 
    int matrix[MAXN][MAXN];//邻接矩阵 
} Graph;

int pathmatirx[MAXN][MAXN];// 路径矩阵，P[v][w]=下一个点
int shortPath[MAXN][MAXN];// 距离矩阵，D[v][w]=v到w的最短路径长度

void short_path_floyd(Graph G, int P[MAXN][MAXN], int D[MAXN][MAXN]){
    int v, w, k;
    // 第一步：初始化 D 和 P 矩阵
    for(v = 0; v < G.vexnum; v++){
        for(w = 0; w < G.vexnum; w++){
            D[v][w] = G.matirx[v][w];
            P[v][w] = w;
        }
    }

    //这里是弗洛伊德算法的核心部分 （三重循环 → 动态规划松弛）
    //k为中间点 尝试以k为中间节点，找更短路径
    for(k = 0; k < G.vexnum; k++){
        //v为起点 
        for(v = 0 ; v < G.vexnum; v++){
            //w为终点 
            for(w =0; w < G.vexnum; w++){
                if(D[v][w] > (D[v][k] + D[k][w])){
                    D[v][w] = D[v][k] + D[k][w];//更新最小路径 
                    P[v][w] = P[v][k];// 更新路径：v先走k，再走后续
                }
            }
        }
    }

    printf("\n初始化的D矩阵\n");
    for(v = 0; v < G.vexnum; v++){
        for(w = 0; w < G.vexnum; w++){
            printf("%d ", D[v][w]);
        }
        printf("\n");
    }

    printf("\n初始化的P矩阵\n");
    for(v = 0; v < G.vexnum; v++){
        for(w = 0; w < G.vexnum; w++){
            printf("%d", P[v][w]);
        }
        printf("\n");
    }

    v = 0;
    w = 3;
    //固定打印：起点0 → 终点3 的最短路径（硬编码）
    printf("\n%d -> %d 的最小路径为：%d\n", v, w, D[v][w]);
    k = P[v][w];
    printf("path: %d", v);//打印起点
    while(k != w){
        printf("-> %d", k);//打印中间点
        k = P[k][w]; 
    }
    printf("-> %d\n", w);
}

int main(){
    int v, w;
    Graph G;
    printf("请输入顶点数:\n");
    scanf("%d", &G.vexnum);
    printf("请输入初始矩阵值：\n");
    for(v = 0; v < G.vexnum; v++){
        for(w = 0; w < G.vexnum; w++){
            scanf("%d", &G.matirx[v][w]);
        }
    }
    printf("\n输入的矩阵值：\n");
    for(v = 0; v < G.vexnum; v++){
        for(w = 0; w < G.vexnum; w++){
            printf("%d ", G.matirx[v][w]);
        }
        printf("\n");
    }
    short_path_floyd(G, pathmatirx, shortPath);
}

和Dijkstra对比

维度 [[2026-06-04-07-Dijkstra最短路径]] Floyd 算法
核心功能 单源最短路径（1 个起点→全顶点） 多源最短路径（任意两点之间）
算法思想 贪心算法（每次选最近的点） 动态规划（中转点松弛）
权值要求 权值必须 ≥ 0（不能有负数） 可含负权值（无负环即可）
时间复杂度 O(V2)（邻接矩阵） O(V3)（三重循环）
空间复杂度 O(V2) O(V2)
代码复杂度 中等（需要标记、更新距离） 极简（三重 for 循环）
适用顶点数量 中大型图（速度快） 小型图（顶点少，速度慢）
你的代码适用场景 导航：从家出发去所有地方 查地图：任意两地之间的距离
