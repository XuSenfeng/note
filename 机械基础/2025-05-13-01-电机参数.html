电机参数

实际的控制需要使用[[2022-9-6-56-CAN协议]]
基础知识在[[2026-05-13-02-物理参数]]

在实际的电机控制的时候发现有几个参数

字节范围 说明 取值映射
Byte6~Byte7 刚度Kp指令 uint16：0~65535 → 0.0~500.0；0x0000=0.0
Byte8~Byte9 阻尼Kd指令 uint16：0~65535 → 0.0~5.0；0x0000=0.0
Byte10~Byte11 前馈力矩指令 uint16：0~65535 → -30.0~30.0Nm；0x8000=0Nm

阻抗控制本质是二阶弹簧 - 阻尼系统，比传统 PID 更适合机器人、外骨骼、柔性伺服

MIT模式

Mixed Integrated Torque为一种混合控制模式，在同一帧CAN数据里包含 位置、速度、扭矩三类 的闭环指令。驱动器里面把位置环、速度环、前馈扭矩相加，得到一个参考电流，然后再交给电流环完成精准扭矩输出

  • kp: 位置比例系数（刚度）, kp = 0 时位置环失效，仅靠速度/扭矩环工作
  • kd: 位置微分系数（阻尼）, kd = 0 时位置环会产生振荡，实际使用时需给一个非零值
  • pos (q): 期望位置, 位置环的目标值
  • vel (dq): 期望速度（单位：rpm）, 速度环的目标值
  • torq (tau): 前馈扭矩（单位：Nm）, 直接给定的扭矩，常用于 纯扭矩控制（kp = kd = 0）

实际使用

  • 匀速转动: kp = 0，kd ≠ 0，pos = 0，vel = 目标速度，torq = 0

只打开速度环，电机以恒定速度运行

  • 纯扭矩输出: kp = 0，kd = 0，pos = 0，vel = 0，torq = 目标扭矩

前馈扭矩直接驱动电流环，适用于 力矩控制（如抓取、阻尼）

  • 点到点位置控制: kp > 0，kd > 0，pos = 目标位置，vel = 0，torq = 0

位置环+速度环共同作用，实现平滑定位

  • 位置‑速度‑扭矩混合: kp > 0，kd > 0，pos = 目标位置，vel = 目标速度，torq = 前馈扭矩

适用于 刚度‑阻尼‑外力补偿（如机械臂的阻抗控制）

三闭环控制模式

所谓三环控制，是指控制环路包含三个控制环路——位置环、速度环、电流环

电流环

如果驱动器单纯运行在电流环方式下（位置环及速度环均不运行，驱动器直接接收外界的电流命令），那么我们就称驱动器运行于力矩模式（对于同一个电机，其产生的力矩$T=电流I*力矩常数Kt$。所以给定了电流就等于给定了力矩）

电流环是伺服电机 FOC 控制里最内层、最基础、响应最快的闭环控制；

专门负责精准控制电机 d/q 轴实际电流，你想让电机输出多大力矩、出力是否平顺、有没有抖动，全靠电流环兜底

以d 轴、q 轴电流指令为给定输入，以电机采样换算出的实际 d/q 轴电流为反馈值，通过 PI 控制器做闭环调节，实时输出 d/q 轴目标电压，再经坐标变换、逆变生成三相驱动电压，从而精准约束、跟踪电机定子电流大小与相位，最终实现电机转矩精准可控的闭环控制系统

速度环

如果驱动器直接运行于速度环方式下（位置环不生效，驱动器直接接收外界的速度命令），那么我们就称驱动器运行于速度模式

电流环是它的一部分或者说是它的一个内环。在探索速度环时，我们假定它的内部的电流环是理想的，即我们要求多大的电流，电流环就真的生成多大的电流；同时我们把电流环复杂的内部运算简化成一个黑箱

左上角有一个速度环的外部输入“速度设定值”，这是速度环的主要外部输入。速度环的运行目标便是让电机的运行速度等于速度设定值。速度环的运行逻辑是，接收到电机编码器传来的电机实际速度，然后用速度设定值减去电机实际速度获得电机速度的误差值，然后改误差值经过PI控制器的运算得到一个电流给定值。该给定值发送给电流环，然后电流环按照我们先前介绍的方式使电机通特定电流，最终电机在该电流的驱动下达到我们的速度设定值

位置环

为速度环再套上一个位置控制环路来行程我们最终的位置环。

位置环接收外界的位置命令，然后用位置命令与编码器反馈的电机实际位置做差得到位置误差。然后该误差经过PI控制器的计算获得速度设定值，然后控制内部的速度环让电机产生对应的速度，最终产生的速度使得电机逼近设定的位置

三种控制模式

转矩控制模式

直接设定电机的输出转矩，是三种模式中最"底层"的控制方式。在这种模式下：

  • 控制原理：通过模拟量输入(如0-10V)或数字通讯直接设定目标转矩值
  • 典型应用：
    • 绕线设备：保持恒张力控制
    • 挤出机：维持恒压力输出
    • 材料测试机：精确控制加载力

速度控制模式

电机转速的精确控制

  • 实现方式：
    • 模拟量输入设定转速
    • 脉冲频率设定转速(PPS控制)
    • 数字通讯直接设定转速值
  • 控制特性：
    • 通常采用PI调节器
    • 需要良好的抗负载扰动能力
    • 动态响应速度介于转矩和位置模式之间

参考输入为速度，速度环输出为力矩环输入

位置控制模式

精准控制电机转角与机械绝对位置，是三种模式中最上层的控制方式

  • 实现方式：
    • 脉冲 + 方向信号输入定位指令
    • 总线 / 数字通讯直接设定目标位置值
    • 支持多轴插补、电子凸轮、轨迹连续运动控制
  • 控制特性：
    • 属于伺服三环最外环，位置环输出作为速度环给定值
    • 定位精度最高，可实现高精度点位停靠与轨迹跟随
    • 动态响应在三种模式中最慢，受位置环、速度环、电流环逐级约束
    • 自带加减速轨迹规划，可平滑启停，避免机械冲击与超调

**$K_{d}$为0的情况下要想控制位置是不可能的，系统会周期震荡。故$K_{p}≠0, K_{d}=0$这种模式是不能用的
  这个系统是一个无阻尼二阶系统，类似于弹簧-质量系统，没有阻尼项，能量无法耗散，导致系统无法稳定

$K_{p}≠0, K_{d}≠0$属于双环反馈系统

参数概念

位置刚度

Kp（位置刚度）：模拟机械弹簧, 把位置误差转换为力矩的比例系数。指令编码为 uint16，线性映射 0.0∼500.00.0∼500.0。0x0000→0.00x0000→0.0。

$T_{p}​=Kp​⋅(θ_{target}​−θ_{real}​)$

Kp 越大
系统刚性越强、位置定位精度越高、跟随误差越小；
但过大会引发震荡、超调、抖动、冲击大，甚至共振。
Kp 越小
系统越柔顺柔软，外力轻轻一推就会偏离目标位置；
适合人机拖拽、避撞、柔性交互，但位置保持精度差。
Kp=0
无位置纠偏力矩，电机完全不受位置闭环约束，随外力自由飘动。

速度阻尼

模拟液压阻尼器 / 减震器，基于速度偏差抑制动态震荡, 把速度误差转换为力矩的比例系数。指令编码为 uint16，线性映射 0.0∼5.00.0∼5.0。0x0000→0.00x0000→0.0。
Kd 偏小
阻尼不足，电机定位后来回晃、低频抖动，启停有余震。
Kd 偏大
抑制震荡效果极好，运动极其平稳；
但过大导致响应滞后、动作迟钝、加减速变慢、电机发热增加。

核心作用：配合 Kp 使用，只调 Kp 不调 Kd，必然震荡。

前馈力矩

直接叠加的偏置力矩，不依赖位置、速度闭环，属于开环力矩给定

直接叠加到控制输出的力矩，编码为 uint16，线性映射 −30.0∼30.0−30.0∼30.0 Nm，0x8000→00x8000→0 Nm。

$τ=Kp​⋅(qcmd​−q)+Kd​⋅(q˙​cmd​−q˙​)+τ_{ff}$

  • Kp 增大：位置更“硬”，跟踪更紧，但更容易抖动、过冲或放大机械间隙/传感噪声。
  • Kd 增大：速度阻尼更强，抑制过冲和振荡，但可能让系统变“闷”、响应变慢。
  • T_{ff}：补偿重力、已知外载或期望输出力矩，减少误差项的负担，提升跟踪质量与效率。
